Help

Please allow microphone permissions to use this page

Here are some commands to try out while recording:

  • "close" (closes the help panel)
  • "color [say a color here]"
  • "help" (opens this help panel)
  • "navigate" (changes pages)
  • "record [text box to populate]"
    • first, middle, last, street, city, state, zip, birthday, notes
  • "reset" (resets the page to white)
  • "stop" (stops recording)
